# asset フォルダについて

このフォルダは、JKパンツ生存戦略診断で使う元素材置き場です。

詳しい用途一覧は以下を参照してください。

```text
docs/04_アセット台帳.md
```

## 現在の使い分け

- `guide-character-front-v2.png`: 扉ページのメインキャラクター
- `generated-types/*.png`: 新規生成したパンツタイプ別の女の子結果ビジュアル
- タコ画像群: 回答後から結果前の「タコパン鑑定中」演出で使用
- `exec-*.png`: 旧/参考用のパンツタイプ別女の子ビジュアル
- 2026年6月27日 01:38 系の画像: 各診断結果のパンツギア画像
- `cf1627fa-*.png`: キャラクター三面図、デザイン参照

## 運用メモ

`src/index.html` は1ファイルSPA要件を守るため、必要な画像を軽量WebP化してBase64埋め込みしています。

そのため、asset内の画像を差し替えただけではアプリ画面は変わりません。素材を更新した場合は、HTMLへの再埋め込みが必要です。

透過PNGは透明背景を保持して使います。RGB変換で透明部分を銀色やグレーに潰さないでください。
