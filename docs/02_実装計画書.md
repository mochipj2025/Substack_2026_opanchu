## 5. 検証コマンド

JavaScript構文チェック:

```powershell
$html = Get-Content -LiteralPath src\index.html -Raw
$matches = [regex]::Matches($html, '<script>([\s\S]*?)</script>')
$matches[$matches.Count - 1].Groups[1].Value | Set-Content -LiteralPath $env:TEMP\opanchu-inline.js -Encoding UTF8
node --check $env:TEMP\opanchu-inline.js
```
