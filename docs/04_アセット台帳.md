## 5. 現在の埋め込み方針

`src/index.html` は1ファイルSPA要件を守るため、必要な画像をWebPへ軽量化したうえでBase64 data URIとしてHTML内に埋め込む。

透過PNG素材はRGBAのままWebP化し、透明背景を保持する。RGB変換で透明部分を銀色・グレー・白へ潰さない。

メリット:

- ファイル単体で配布できる
- 画像パス切れが起きない
- GitHub Pagesやローカル直開きでも動かしやすい

注意点:

- HTMLファイルサイズは大きくなる
- アセット差し替え時は再埋め込みが必要
- 透過素材を再埋め込みする場合は、アルファ保持を必ず確認する
